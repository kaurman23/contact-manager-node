# contact-manager-node
A web application which manages contacts.

## Build
Created using HTML/CSS, JQuery, NodeJS, Express and EJX with mongoDB as a database. Deployed using heroku.

## Check it out here
https://fierce-wave-82294.herokuapp.com/

## Properties
 * Create Contacts
 * Delete Contacts
 * Search Contacts
